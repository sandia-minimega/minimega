What is minimega?

minimega is a tool for launching and managing virtual machines. It can run distributed across a cluster, or on a laptop. minimega is fast, easy to deploy, and can scale to run across massive clusters.

OpenStack and other VM-management tools need lots of configuration and they're still slow! minimega requires no configuration beyond a few optional command line arguments, and it launches VMs fast. In seconds, you can launch and start as many VMs as your physical memory can hold--but thanks to minimega's KSM integration, duplicated memory will be compressed and you can start even more VMs.

The minimega repository also includes several more tools and libraries to make VM creation and cluster management easier

Getting Started

The Installation Guide contains instructions for fetching and installing minimega. Once you've installed it, the User's Guide will help you get started using minimega. It also provides online help for individual commands.
